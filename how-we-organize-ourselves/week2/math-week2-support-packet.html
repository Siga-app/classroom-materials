P5 | Decision Checklist

Should I Use GCF or LCM?

Use this page before solving each problem.
Name:
Date:

Use GCF when...

The problem is about dividing or sharing.
The groups, kits, bags, or bundles need to be equal.
Nothing should be left over.
The problem asks for the greatest number of equal groups.

Use LCM when...

The problem is about schedules, cycles, or patterns repeating.
Two things happen every ___ units of time.
The problem asks when both things happen together again.
The problem asks for the first or earliest common time.
